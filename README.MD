# Counter MVI

Простое приложение-счетчик, демонстрирующее чистую реализацию архитектуры MVI (Model-View-Intent) на Android с использованием Kotlin и Jetpack Compose.

## Основные концепции

Приложение построено на принципах одностороннего потока данных, где каждое пользовательское действие проходит через четкий цикл: Intent → Reducer → State → UI.

## Ключевые особенности

- **Предсказуемое состояние** - В любой момент времени состояние интерфейса полностью определено и неизменно
- **Чистая бизнес-логика** - Вся логика обработки сосредоточена в редукторах, которые легко тестировать
- **Разделение ответственности** - Четкое разделение между UI, управлением состоянием и бизнес-логикой
- **Эффекты как события** - Отдельный механизм для одноразовых событий (тосты, навигация)
- **Реактивная архитектура** - Использование Kotlin Coroutines и Flow для асинхронных операций

## Функциональность

Приложение позволяет:
- Увеличивать и уменьшать значение счетчика
- Сбрасывать счетчик к нулю
- Быстро изменять значение на ±10
- Просматривать историю операций с временными метками
- Очищать историю операций
- Отслеживать статистику использования
- Получать уведомления при достижении граничных значений

## Технологический стек

- Kotlin
- Jetpack Compose (декларативный UI)
- Kotlin Coroutines & Flow (асинхронность)
- Koin (внедрение зависимостей)
- MVI (архитектурный паттерн)

Этот проект служит образовательным примером для разработчиков, изучающих современные подходы к построению Android-приложений с акцентом на чистую архитектуру и предсказуемое управление состоянием.
